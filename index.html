About

Flight Agent is yet another home/professional cockpit builder application. Show instruments, moving map, remote mbile application, …

Web based

Run it on any modern browser, android, rpi, windows, linux, …

Low CPU & RAM

Server app normally uses 10Mb of RAM and CPU bellow 0.5%

Mobile remote application

Control the simulator from your mobile. Set failures, …

Instrument editor

Create instruments and move/scale them

Professional use

Session management for professional simulator

Moving map

Control ariplane position with OSM maps

FSX, P3D, MSFS2020

Compatible with Microsoft Flight Simulators

Cockpit hardware

Connect hardware to the system

One executable

Just one exe has everything

Get started

  1. Download

Download a single executable

  1. Execute

Execute it. The app tries to know what is inside the computer an creates an initial config.yaml file for you.

2a. Configure to start with Windows (Optional)

Next time starting the computer the program will start automatically

  1. Open a web browser

Manually or automatically open http://localhost:8080. It is automatically open if a config.yml file is autogenerated

  1. Get FA Remote and cofmgure it to the server

View flight data remotelly and set failure, weather, etc…
