About

Flight Agent is an application to extent flight simulators. Show panel instruments, show a moving map, control leds, buttons and screens, … see below

Web based

Run it on any modern browser, android, rpi, windows, linux, iPad, …

Low CPU & RAM

Server app normally uses 10Mb of RAM and CPU bellow 0.5%

Mobile remote application

Control the simulator from your mobile. Set failures, …

Instrument editor

Create instruments and move/scale them

Professional use

Session management for professional simulator, contact flightagent.pro@gmail.com

Moving map

Control ariplane position with OSM maps

FSX, P3D, MSFS2020

Compatible with Microsoft Flight Simulators

Cockpit hardware

Connect hardware to the system

One executable

Download just one exe an execute it

Customizable

Easy interact with, events & actions, openAPI, …

Backups

Easy backups of configuration files, how many times did you lost the sim config?

Multi-platform

The server works on windows, linux, mac

Get started

  1. Download

Download a single file (for FSX the 32bits version, for MSFS2020 the 64 bits version)

  1. Execute

Execute it. A default (looking at your computer) configuration will be created

  1. Open a web browser

From the same computer of the server open http://localhost:8080
