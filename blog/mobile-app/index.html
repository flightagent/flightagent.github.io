Flight Agent remote application

Application to control remotely the simulator and cockpit around it. Application can share with a QR a public URL with the current flight, to share it to customers.

  • Configuration
  • Flight settings
  • Sessions

Configuration inside modules:

  temperatures:
    - temperature:
      name: <name>
      url: http://<api>:8080/api/v1/weather
    - temperature:
      name: <name>
      url: http://<ip>:8080/api/v1/weather

  famobile:
    failures:
      - failure:
        name: Master off
        event: setsimvar/master=0
      - failure:
        name: Mixture off
        event: setsimvar/mixture=0
      - failure:
        name: Left tank empty
        event: setsimvar/lefttank=0
      - failure:
        name: Right tank empty
        event: setsimvar/righttank=0
      - failure:
        name: Left tank Full
        event: setsimvar/lefttank=80
      - failure:
        name: Right tank Full
        event: setsimvar/righttank=80
    messages:
      - message:
        name: Takeoff rnw12
        event: Despegue por la 12
      - message:
        name: Land rnw12
        event: Aterrize por la 12
    views:
      - view:
        name: internal
        event: viewinternal
      - view:
        name: external
        event: viewexternal
    cloudserver:
      ip: <ip server>
      apikey: <apikey>
    mumbleserver:
      ip: <ip>
      port: 64738
      username: fagent
    cameras:
      - rtsp://<ip 1>:8554/stream
      - rtsp://<ip 2>:8554/stream
      - ... other
